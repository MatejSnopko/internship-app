Follow these steps on Google Cloud. You only do this once.

  1. Go to console.cloud.google.com → sign in with your Gmail.
  2. Click the project dropdown (top-left) → New Project → name it anything → Create.
  3. Search for Gmail API in the top bar → click it → click Enable.
  4. Go to APIs & Services → OAuth consent screen → choose External → Create.
  5. Fill in App name + your email → Save and Continue through all steps.
  6. Go to Test users → + Add Users → add your Gmail → Save.
  7. Go to APIs & Services → Credentials → + Create Credentials → OAuth client ID.
